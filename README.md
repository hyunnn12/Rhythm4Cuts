# 💬 프로젝트 배경

WebRtc를 활용해 온라인으로 서로 "노래"를 부르면서 즐거운 시간을 보내고, 그 즐거운 순간의 기록을 "네컷사진"으로 저장하여 추억을 남기는것을 어떨까?
